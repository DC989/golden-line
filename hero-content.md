## Design Notes for Hero Section

**Recommended Layout:**
- Full-width hero with background image (snowy mountain or sleek vehicle)
- Text overlay with semi-transparent dark background for readability
- Headline positioned top-left or center
- Subheading below headline
- Description text below subheading
- Two CTA buttons side-by-side (or stacked on mobile)

**Color Scheme (Gold Palette):**
- Primary CTA Button: Gold (#D4AF37 or similar metallic gold)
- Secondary CTA Button: Dark Navy/Black with gold border
- Text: White on dark overlay
- Subheading: Light gold accent color

**Button Styling:**
- Primary button: Bold, larger, gold background with shadow
- Secondary button: Outline style with gold border
- Hover state: Slight scale increase, glow effect
- Both buttons include icons (calendar + arrow for "Book Now", phone icon for "Call")

**Responsive Behavior:**
- Desktop: 2-column layout with buttons side-by-side
- Mobile: Full-width buttons stacked vertically
- Headline font size: Scale down on mobile
- Description text: Slightly smaller on mobile

**Animation Suggestions:**
- Fade-in on page load
- Subtle parallax effect on background image (if desktop)
- Button hover: Color shift, shadow expansion
- Optional: Animated arrow pointing downward to encourage scrolling
